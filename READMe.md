# n8n-youtube-animation-automation

Koleksi workflow **n8n** open-source untuk creator YouTube Shorts & AI Animation Automation.

## 🎯 Tujuan Proyek
Membantu creator (terutama pemula di niche animasi, motivasi, edukasi) mengotomatisasi produksi konten YouTube Shorts menggunakan n8n + yt-dlp + AI (Gemini/Claude) + ffmpeg.

## 📁 Workflow yang Sudah Ada

- **AI Clipper v4** (`clipper-v4-final.json`)  
  yt-dlp + Gemini analysis + auto clipping

- **AI Clipper Simple** (`clipper-simple.json`)  
  Versi ringan pakai RSS

- **DAZN Campaign Clipper** (`dazn-fury-hall.json`)  
  Contoh workflow khusus event/brand campaign

- **AI Clipper FINAL** (`n8n-clipper-final.json`)  
  Paling advance: RSS + Keyword Search + Google Sheet deduplication + Gemini + siap auto upload

## 🚀 Cara Pakai
1. Import JSON workflow ke n8n
2. Sesuaikan API Key (Gemini, Telegram, dll)
3. Jalankan

## Rencana Pengembangan Selanjutnya
